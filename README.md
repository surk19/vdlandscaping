# VD Landscape Pros Website

Static HTML/CSS/JS website for GitHub Pages.

Pages included:
- Home
- Services
- Portfolio
- Contact

Before publishing:
1. Replace placeholder images with real project photos.
2. Replace the contact form email in contact.html if needed.
3. Update service areas if you want city-specific SEO.
4. Add Google Business Profile link once available.

GitHub Pages setup:
1. Create a GitHub repository named vdlandscapepros or any name you prefer.
2. Upload all files from this folder to the repository root.
3. Go to Settings > Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select main branch and /root folder.
6. Save.
7. In your domain registrar DNS, point vdlandscapepros.com to GitHub Pages.

Typical DNS records for GitHub Pages:
A records for @:
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153

CNAME for www:
yourgithubusername.github.io

Then add vdlandscapepros.com inside GitHub Pages custom domain.
